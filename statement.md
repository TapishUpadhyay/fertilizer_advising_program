Problem :
Small-scale farmers and home gardeners often struggle with identifying soil nutrient deficiencies and selecting appropriate fertilizers, leading to:
Poor Crop Yields: Incorrect fertilizer application resulting in suboptimal plant growth

Scope 
 Basic NPK (Nitrogen, Phosphorus, Potassium) deficiency identification
 General fertilizer recommendations based on primary nutrient needs
 helps improve education
 Simple, intuitive user interface for non-technical users
 Desktop application with offline functionality

 Target Users
 Farmers who have less knowledge or dealing with the same problem

Features
as mentioned previously
Primary Nutrient Deficiency Identification
nutrient options (N, P, K)
Visual color-coded selection system
Simple interface
deficiency analysis
